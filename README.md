# 01 🏥 全流程医院信息系统 (Hospital Information System)

一个现代化的、面向医生端的全流程医院信息系统，覆盖从患者挂号到费用结算的核心业务。

## 📌 核心功能
- **医生认证**：安全的密码登录与会话管理。
- **数据看板**：实时展示总建档人数、当日就诊/检查人数，并通过 ECharts 可视化近14天就诊趋势、医嘱类型分布及科室热度。
- **患者管理**：支持新增、编辑患者基本信息。
- **智能病历**：结构化表单书写，集成 ICD-10 诊断编码的多维度（名称/拼音/编码）智能搜索。
- **医嘱开立**：从预设字典选择药品/检查，自动计算总价。
- **费用结算**：汇总所有医嘱费用，提供移动端友好的支付确认界面。

## 🛠 技术栈
- **后端**: Java, Spring Boot, MySQL
- **前端**: Vue 3, Vuex, Element Plus, ECharts, Axios
- **架构**: 前后端分离

## 💡 亮点
- **用户体验优先**：密码可见性切换、ICD编码模糊搜索、病历一键打印。
- **工程化实践**：模块化组件设计、Vuex 状态管理、Axios 请求封装。
- **业务闭环**：完整复现了真实医院的核心工作流。


# 02 🏥 Medi  (C# WinForms)

一个功能完备的 Windows 桌面版医院信息管理系统，采用 C# 和 WinForms 开发，为医生提供本地化的诊疗辅助工具。

## 📌 核心功能
- **用户认证**：安全的账号密码登录。
- **患者管理**：支持新增、编辑、查询患者基本信息（姓名、性别、年龄、联系方式等）。
- **电子病历**：结构化录入主诉、现病史、既往史、体格检查、初步诊断等内容。
- **医嘱管理**：从预设药品/检查字典中选择项目，系统自动计算总费用。
- **费用结算**：清晰展示所有医嘱明细及总价，支持打印结算单。
- **数据持久化**：所有数据存储于本地 SQLite 数据库，确保离线可用性。

## 🛠 技术栈
- **语言**: C#
- **框架**: .NET Framework, WinForms
- **数据库**: SQLite
- **开发工具**: Visual Studio

## 💡 亮点
- **用户体验**：界面简洁直观，操作符合医生工作习惯。
- **工程实践**：实现了完整的 CRUD（增删改查）操作和数据验证逻辑。
- **技术演进**：此项目为后续 Web 版 HIS 的开发奠定了业务逻辑基础。
