# 🏥 全流程医院信息系统 (Hospital Information System)

一个现代化的、面向医生端的全流程医院信息系统，覆盖从患者挂号到费用结算的核心业务。

## 📌 核心功能
- **医生认证**：安全的密码登录与会话管理。
- **数据看板**：实时展示总建档人数、当日就诊/检查人数，并通过 ECharts 可视化近14天就诊趋势、医嘱类型分布及科室热度。
- **患者管理**：支持新增、编辑患者基本信息。
- **智能病历**：结构化表单书写，集成 ICD-10 诊断编码的多维度（名称/拼音/编码）智能搜索。
- **医嘱开立**：从预设字典选择药品/检查，自动计算总价。
- **费用结算**：汇总所有医嘱费用，提供移动端友好的支付确认界面。

## 🛠 技术栈
- **后端**: Java, Spring Boot, MySQL
- **前端**: Vue 3, Vuex, Element Plus, ECharts, Axios
- **架构**: 前后端分离

## 💡 亮点
- **用户体验优先**：密码可见性切换、ICD编码模糊搜索、病历一键打印。
- **工程化实践**：模块化组件设计、Vuex 状态管理、Axios 请求封装。
- **业务闭环**：完整复现了真实医院的核心工作流。
